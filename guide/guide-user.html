1. 접속 및 로그인

1-1. 웹 주소

스마트폰 브라우저(Safari, Chrome 권장)에서 다음 주소로 접속합니다.

https://YOUR-S3-BUCKET.s3.ap-northeast-2.amazonaws.com/index.html

1-2. 로그인 절차

  1. 본인 이름을 입력합니다 (자동완성 지원).
  2. 비밀번호를 입력합니다. 최초 비밀번호는 abc123!.
  3. 입장하기 버튼을 누릅니다.
⚠ 첫 로그인 직후 비밀번호를 변경해 주세요
탑바 우측 ⋯ 메뉴 → 비번변경.

2. 메인 대시보드

로그인 후 첫 화면은 현재 업무 / 다음 내 업무 두 개의 카드로 구성됩니다.

2-1. 현재 업무 카드 (빨간색)

  • 현재 진행 중인 시간대의 일정명과 본인 담당 업무가 표시됩니다.
  • 하단 남은 시간 막대로 진행 상황 시각화.
  • 본인 업무를 마치면 완료 표시 체크박스 클릭.
  • 카드 자체를 누르면 상세 가이드(위키)가 열립니다.

2-2. 다음 내 업무 카드 (남색)

  • 본인이 담당으로 등록된 다음 시간대를 보여줍니다.
  • "X시간 Y분 후 시작" 카운트다운 표시.
  • 이 카드도 눌러서 상세 가이드를 미리 확인할 수 있습니다.

2-3. 상단 진행률 바

1일차/2일차 전체 일정 중 현재 어느 시간대인지 표시.

3. 상세 가이드 (위키)

현재 업무 카드 또는 다음 내 업무 카드를 누르면 시트가 열려 해당 시간대의 상세 가이드를 볼 수 있습니다.

3-1. 시트 구성

  • ★ 내 업무: 본인 직책과 담당 업무 강조.
  • 상세 가이드: 일정 흐름 요약 + 단계별 행동 + ⚠ 주의사항.
💡 활용 팁
본인 업무가 시작되기 전에 다음 내 업무 카드를 미리 눌러서 가이드를 읽어두면 당황하지 않고 진행할 수 있습니다.

4. 전체 일정 보기

대시보드 하단의 전체 일정 보기 ❯ 버튼을 누르면 1일차/2일차 전체 타임테이블이 펼쳐집니다.

4-1. 보기 옵션

  • 일반 사용자: 본인 담당 업무만 강조 (담당 없는 시간대는 회색).
  • 현재 시간대는 "지금" 빨간 라벨로 표시.

4-2. 일자 전환

1일차 / 2일차 탭을 눌러 전환합니다. 행사 당일 자동으로 해당 일자가 활성화됩니다.

5. 비밀번호 변경

  1. 탑바 우측 ⋯ 메뉴를 누릅니다.
  2. 비번변경 선택.
  3. 현재 비밀번호 → 새 비밀번호 → 확인 순으로 입력.
  4. 변경하기 버튼 클릭.
🔒 비밀번호 보안
비밀번호는 SHA-256 해시로 변환되어 저장되며, 서버에는 평문이 저장되지 않습니다. 분실 시 어드민에게 초기화 요청하세요.

6. 유의 사항 / FAQ

Q. 로그인 직후 "비밀번호가 맞지 않습니다"가 뜹니다.

서버 동기화 직후 다시 시도하세요. "서버 연결 중입니다" 메시지가 뜨면 1-2초 대기.

Q. 비밀번호를 잊었습니다.

기획국 Member05에게 초기화 요청.

Q. 다른 기기에서 같은 계정으로 로그인할 수 있나요?

가능합니다. 계정·일정·공지·할일 모두 5초 단위로 자동 동기화.

Q. 인터넷이 잠시 끊기면?

탑바에 "오프라인" 표시가 뜨고, 마지막 동기화된 데이터로 계속 표시. 다시 연결되면 자동 복구.

Q. 본부석에 보고해야 할 일이 생기면?

해당 시간대의 상세 가이드 하단 ⚠ 주의사항을 참고하고, 즉시 학생회장(Member36) 또는 부학생회장(Member04)에게 연락.

⚠ 행사 당일 가장 중요한 3가지
  1. 본인 조 인원 누락 즉시 보고
  2. 안전사고 (구토자, 부상자) 즉시 본부석
  3. 강압적 음주 게임 절대 금지
